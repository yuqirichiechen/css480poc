Keyboard Access

This site is fully navigable by keyboard. Here is how to get around:

KeyAction
TabMove forward through all links, buttons, and interactive elements
Shift + TabMove backward through interactive elements
Enter or SpaceActivate the focused button or link
↑ / ↓ Arrow keysMove between items in the interests list when it is focused
Home / EndJump to the first or last item in the interests list
Enter on a nav linkNavigate to Home or Things to Read page

The most important usability principle applied here is focus visibility — every interactive element shows a clear, high-contrast blue outline when focused, so keyboard users always know exactly where they are on the page. A "Skip to main content" link is also the very first focusable element, letting users jump past the navigation bar without tabbing through every link.
